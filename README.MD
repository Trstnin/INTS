# Into Startups (INTS) 🚀

A comprehensive platform connecting startup enthusiasts with stories, resources, and learning opportunities.

## 📋 Project Overview

Into Startups is a web platform designed to:
- Share inspiring startup stories
- Provide detailed startup reviews
- Offer learning resources for entrepreneurs
- Build a community of startup enthusiasts

## 🏗️ Project Structure

```
INTS/
├── Frontend/           # React + Tailwind CSS frontend
│   ├── src/           # Source files
│   └── README.md      # Frontend documentation
├── Backend/           # (Coming soon)
│   └── README.md
└── README.md          # Main project documentation
```

## 🛠️ Technology Stack

### Frontend
- React.js
- Tailwind CSS
- React Router DOM
- Preline UI Components

### Backend (Planned)
- Node.js
- Express
- MongoDB
- JWT Authentication

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd INTS
   ```

2. Setup Frontend:
   ```bash
   cd Frontend
   npm install
   npm run dev
   ```
